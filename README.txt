To run just type python main.py in console.
Additionas features:
-additional buffer display on the top of the calculator
-repeating last action by clicking "="  (e.g. "2+2===" equals eight)
-chaining operations, one after another without need to click "=" each time (e.g. "2+6/2=" (this is not an equation! This is the order of the pressed keys) equals four ())
-changing operator after clicking one (e.g. "6+-/3=" equals 2)
-creating negative numbers (if you typing it as a first number, for the next one you can just use "-" instead of "+" when you want to add a negative number, so it's not necessary)
-clearing entry (CE)
-reseting calculator (C)
-comma
